Unicode 引用

1. 添加 @font-face 块, 引用图标字体文件；

@font-face {
  font-family: 'iconfont';
  src: url('iconfont.eot');
  src: url('iconfont.eot?#iefix') format('embedded-opentype'),
  url('iconfont.woff2') format('woff2'),
  url('iconfont.woff') format('woff'),
  url('iconfont.ttf') format('truetype'),
  url('iconfont.svg#iconfont') format('svg');
}

2. 设置 .iconfont 的基本样式；

.iconfont {
  font-family: "iconfont" !important;
  font-size: 16px;
  font-style: normal;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}

3. 应用图标，输入对应的 Unicode 值。

<span class="iconfont">&#xe624;</span>

"iconfont" 是项目下的 “iconfont family” 字段，可以通过编辑项目查看，默认是 "iconfont"。

Class 引用

1. 引入已经将 unicode 与类名一一对应的 iconfont.css 文件；

<link rel="stylesheet" href="./iconfont.css">

2. 应用图标，使用对应的类名。

<span class="iconfont prefix-classname"></span>

"iconfont" 是项目下的 “iconfont family” 字段，可以通过编辑项目查看，默认是 "iconfont"。
"prefix-" 是项目下的“iconfont 前缀”字段，可以通过编辑项目查看，默认是 "icon-"。

Symbol 引用

1. 引入 iconfont.js 文件，向页面添加 SVG sprite；

<script src="./iconfont.js"></script>

2. 设置基本样式；

<style>
.icon {
  width: 1em;
  height: 1em;
  vertical-align: -0.15em;
  fill: currentColor;
  overflow: hidden;
}
</style>

3. 应用图标，使用对应的 id 值。

<svg class="icon" aria-hidden="true">
<use xlink:href="#prefix-xxx"></use>
</svg>

"iconfont" 是项目下的 “iconfont family” 字段，可以通过编辑项目查看，默认是 "iconfont"。
"prefix-" 是项目下的“iconfont 前缀”字段，可以通过编辑项目查看，默认是 "icon-"。
